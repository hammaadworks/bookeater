# Project Instructions

## Documentation Standards
- **Rule**: The `docs/` folder must always be kept updated with current implementation and design decisions.
- **ADRs**: Use `docs/adr/` for documenting significant architectural choices.
- **Agent Context**: Refer to `AGENTS.md` for specific AI agent instructions and project workflows.

## Development Workflow
- **Tech Stack**: React 18 (TypeScript), Vite, Tailwind CSS.
- **Persistence**: Local-first via IndexedDB (Dexie.js).
- **Tests**: Use Vitest for all unit and integration tests.
